# MVP Roadmap

## Phase 1: Core Architecture Setup
- Initialize repository structure and documentation.
- Build backend scaffold, auth API, database connection, and socket server.
- Create Unity startup scripts and service registration pattern.
- Define roadmap and folder conventions.

## Phase 2: Character Controller + Combat
- Implement player movement, parkour, sprinting, climbing.
- Build hand-to-hand melee combat manager, combo chains, blocking/parrying.
- Add animation support, input mapping, combat state machine.

## Phase 3: Multiplayer Systems
- Add matchmaking lobby, socket room management, real-time sync.
- Implement client prediction and network reconciliation scaffolding.
- Create basic multiplayer session flow.

## Phase 4: Backend + Authentication
- Add user account registration and login.
- Build JWT security and session validation.
- Add player profile, friend/clan service scaffolding.

## Phase 5: Character Creator
- Add modular customization data model.
- Build Unity creator UI and preset serialization.
- Sync cosmetic loadout metadata through backend.

## Phase 6: Open-World Systems
- Add city world data, NPC spawners, dynamic weather and day/night cycles.
- Build mission zone and interactive object systems.

## Phase 7: AI Systems
- Implement enemy combat AI, police/gang/civilian state machines.
- Add utility AI and behavior tree framework.

## Phase 8: Economy + Quests
- Add currency, inventory, shops, marketplace, loot, reward systems.
- Add story quest service, side missions, reputation and dialogue.

## Phase 9: Optimization
- Optimize network traffic, asset streaming, server scaling, and performance.
- Add telemetry, analytics, stress tests.

## Phase 10: Deployment + Live Service Setup
- Create production CI/CD workflows.
- Deploy backend container stacks and database resources.
- Setup live event service, cloud save, and analytics pipelines.
