# Project Structure

## Top-level folders

- `UnityClient/`: Unity project container, game assets, scripts, scenes, systems, UI, and data.
- `Backend/`: Node.js backend service, API, WebSocket server, database schema, Docker build configuration.
- `Shared/`: Cross-platform contracts and shared enums, DTOs, type definitions, network message schemas.
- `Database/`: SQL schema, migration plans, seed data, entity relationship diagrams.
- `Docs/`: Design documentation, architecture guides, development roadmaps, technical decisions.
- `DevOps/`: CI/CD manifests, Docker Compose examples, deployment notes, environment templates.
- `.github/`: GitHub Actions workflows, issue and PR templates.
